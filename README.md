# MOVIES WEBSITE

Movies Website is a React App where you can find movies, tv series or actors.
**Live: ** https://movieswebsite.netlify.app/
## Installation
```bash
npm install
npm start
```
## Stack
- React
- Redux
- Redux Thunk
- Axios
- StyledComponents
- React Router
